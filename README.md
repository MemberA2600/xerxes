<h2>Xerxes - A game created in Winteracter Fortran</h2>
<br>
<p style="text-align:justified" >You may ask, why would anyone do this? I would answer: <b>Why do nobody do this? :D </b></p>
<p style="text-align:justified" >The project is in a really early stage, but the plans are:</p>
<ul>
<li>A classic 4:3 screen up to 2048 * 1536, with 256 colors (based on the 16 Spectrum colors, but extended more depth)!</li>
<li>TIA (Atari 2600) emulation for sound effects!</li>
<li>YM3812 (Adlib) emulation for music!</li>
<li>Classic JRPG / Visual Novel, arcanoid + danmaku shooter game hybrid!</li>
</ul>
